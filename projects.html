Project

Het project waar ik aan mocht werken was een CRM (Customer Relation Manager), hierin kwamen alle klanten en de bijbehorende projecten. Waar het vooral voor diende was het helpen van de sales afdeling bij 23G zodat zij sneller een overzicht hadden van welke versies een bepaalde project gebruikt. Als een product een x aantal versies achter loopt kan dit verkocht worden, momenteel worden de versies niet live bij gehouden en dit moest dus veranderen. Aan de hand van de GitHub API worden alle repositories opgehaald en hier kunnen versies uit gelezen worden.
