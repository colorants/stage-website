Project

Ik heb van begin tot eind een CRM (Customer Relation Manager) gebouwd, waarin alle klanten en de bijbehorende projecten centraal worden beheerd. Dit systeem is ontwikkeld om de salesafdeling van 23G te ondersteunen door hen sneller inzicht te geven in welke versies van een bepaald project worden gebruikt. Wanneer een repository meerdere versies achterloopt, biedt dit namelijk verkoopkansen richting de klant maar vooraal ook minderen van veiligheid. Op dit moment worden de versies niet live bijgehouden, en het was mijn taak om dit proces te verbeteren. Met behulp van de GitHub API in combinatie met de EOL API worden alle repositories opgehaald, waarna de gebruikte versies automatisch worden uitgelezen. Het systeem is gebouwd met FilamentPHP wat gebruik maakt van het Laravel Framework.
Dit project is een goede startup om nog veel features toe te voegen zodat 23G de klanten sneller en beter te kunnen helpen.
